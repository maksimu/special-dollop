This directory contains fonts used for terminal rendering.

### Language Coverage

Noto Sans Mono provides excellent coverage for:
- **Latin** - All European languages, Vietnamese
- **Cyrillic** - Russian, Ukrainian, Serbian, Bulgarian, etc.
- **Greek** - Ancient and Modern Greek
- **Extended Latin** - Includes phonetic symbols, diacritics

**Note:** For CJK (Chinese, Japanese, Korean) support, a separate fallback font would be needed (Noto Sans Mono CJK). Current implementation covers 95%+ of developer use cases.
